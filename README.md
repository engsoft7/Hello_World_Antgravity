# 🌌 Hello World Antgravity

Uma página **Hello World** moderna com animações visuais impressionantes, construída com HTML e CSS puro.

## ✨ Features

- 🎨 **Texto com gradiente animado** — transições suaves entre roxo, rosa e ciano
- 🔮 **Orbs flutuantes** — esferas com desfoque que se movem no fundo escuro
- 💫 **Anéis pulsantes** — efeito de pulso expandindo ao redor do texto
- 📱 **Design responsivo** — adapta-se a qualquer tamanho de tela
- 🖋️ **Tipografia moderna** — fonte Inter do Google Fonts

## 🚀 Como usar

Basta abrir o arquivo `index.html` no seu navegador:

```bash
# Clone o repositório
git clone https://github.com/engsoft7/Hello_World_Antgravity.git

# Abra no navegador
open index.html
```

Ou acesse diretamente pelo file explorer — nenhuma dependência necessária!

## 🛠️ Tecnologias

| Tecnologia | Uso |
|------------|-----|
| HTML5 | Estrutura semântica |
| CSS3 | Animações, gradientes, glassmorphism |
| Google Fonts | Tipografia (Inter) |

A página apresenta um fundo escuro com orbs coloridos flutuantes e o texto "Hello, World!" em gradiente animado, acompanhado do subtítulo "Bem-vindo ao universo".
